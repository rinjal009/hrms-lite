HRMS Lite is a lightweight Human Resource Management System built as a full-stack web application.  
It allows an admin user to manage employee records and track daily attendance through REST APIs and a simple web interface.

The goal of this project is to demonstrate core full-stack development skills including backend API design, database persistence, frontend integration, and deployment readiness.

Main features include:
- Add employees
- View employee list
- Delete employees
- Mark daily attendance (Present / Absent)
- View attendance records
- RESTful API with validation and proper error handling
